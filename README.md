# appage
A page for your app

To rebuild the app list, run:

```shell
ruby appage.rb
```

Thanks to [@gamerlv](https://github.com/gamerlv) for [the script](https://gist.github.com/gamerlv/4bb5e59415f239e8c79ff1d473e54520)!
